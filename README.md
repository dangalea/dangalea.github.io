# dangalea.github.io

Personal website for Daniel Galea, PhD - Machine Learning Scientist

## Features

- Modern, clean design with glassmorphism effects
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations
- Single-page application with section navigation
- Optimized for GitHub Pages

## Structure

- `index.html` - Main HTML file
- `assets/css/style.css` - Modern CSS with custom properties
- `assets/js/script.js` - JavaScript for interactivity
- `assets/images/` - Images and favicon
- `resume_2024.pdf` - Downloadable resume

## Technologies

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Intersection Observer API for scroll animations

## GitHub Pages

This site is configured for GitHub Pages deployment. The `.nojekyll` file ensures all files are served correctly.
